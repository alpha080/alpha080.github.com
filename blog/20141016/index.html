我所用到的 Sublimetext 插件

Sublimetext 是一个非常强大的编辑器，它的一些特性非常牛，比如 gotoanything, multiple selections, command palette. 不过也有一些不够 好的地方，比如在 Linux 下汉字输入就要麻烦一点。由于工作中需要在 U 盘中装个图形界面编辑器( Windows xp 下)，而且 gvim 在U 盘下确实 不如 Sublimetext，所以就有了今天的文章。

我所用的 Sublimetext3 的插件有：

  1. Package Control
  2. git
  3. emmet
  4. SublimeCodeIntel
  5. Alignment
  6. Angularjs
  7. BracketHighlighter
  8. ColorHighlighter
  9. Inc-Dec-Value
  10. Modific
  11. Side Bar
  12. TabsExtra
  13. Tag
  14. Sublimelinter
  15. Arduino-like IDE
  16. Side Bar Enhancements
  17. ColorPicker
  18. Cheat Sheets
  19. Jquery
  20. Html5
  21. Vintageous

什么时候也试试看 github 的 atom 编辑器跟 adobe 的 bracket 编辑器，好像它们也有很多吸引人的特性。所以我应该感谢老天把我生在这个编辑 器百花齐放的时代么？
